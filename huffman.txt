FUNCION HuffmanTree:
INPUT: MinHeap<BinaryTree> / PriorityQueue<BinaryTree>
OUTPUT: BinaryTree
USAGE: BinaryTree tree = huffmanTree(heap)
BEGIN
	BinaryTree tree;
	WHILE (heap.size() != 1)
			t1 = heap.deleteMin()
			t2 = heap.deleteMin()
			heap.insert(t(t1,t2))
		END
	return heap.findMin()
END
